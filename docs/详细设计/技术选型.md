# 技术选型指引

| 层次 | 推荐方向 | 说明 |
|---|---|---|
| Hermes 框架 | Electron / Tauri / Web | 桌面应用或 Web 应用 |
| Hermes AI | Claude API / 本地大模型 | 设计态推理 |
| Bastion 引擎 | ONNX Runtime / TensorRT / TFLite | 执行 Hermes 下发的推理模型 |
| Bastion HAL | C / 汇编 / Rust | 传感器/操作器型号驱动 |
| Terminal 固件 | C/C++ / Rust | 实时性与资源效率 |
| 通信协议 | MQTT / gRPC / 自定义二进制 | 根据场景选择 |
| 建模语言 | SysML / 自定义 DSL | 系统模型定义 |
| 仿真 | 开源 CAE 工具链 | CAE 自动化脚本驱动 |
