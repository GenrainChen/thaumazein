# 通信设计

```
┌──────────┐     ┌──────────┐     ┌──────────┐
│  Sensor   │────→│ Bastion  │────→│ Hermes   │
│ Terminal  │     │ (Hub)    │     │          │
├──────────┤     ├──────────┤     ├──────────┤
│ Operator  │←───│ Bastion  │←───│ Hermes   │
│ Terminal  │     │ (Hub)    │     │          │
├──────────┤     │          │     │          │
│感知-操作  │←──→│          │←──→│          │
│一体终端   │     │          │     │          │
└──────────┘     └──────────┘     └──────────┘

  底层(物理)       中间层+高层        高层
```

Bastion ↔ Terminal：寄存器级读写，HAL 抽象后为"传感器轮询""操作器命令"，不含语义。

Hermes ↔ Bastion：行为包部署、遥测流、控制指令、经验数据、心跳，详见 [架构设计](../架构设计.md) 通信拓扑章节。
